# Hospital_managemnt_oop
oop project for hospital management system
This is a C++ console-based program that allows managing patients, prescriptions, and medicines using text files. 
Key points:  
- Demonstrates basic object-oriented programming concepts.  
- Manage patients: add, edit, delete, search, and display records.  
- Manage medicines: add, edit, delete, and display inventory.  
- Manage prescriptions: add, edit, delete, and view prescriptions.  
- Text files (patients.txt, Medicines.txt, prescriptions.txt) are created and updated automatically.  
- To run: open in any C++ IDE (e.g., Dev C++ or Code::Blocks), compile OOPproject.cpp, and follow the console menu.
